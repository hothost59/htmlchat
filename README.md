# htmlchat
Unblocked HTML chat. Made for people wanting to chat at school.
# Usage
First, go to https://console.firebase.google.com/, and login. Then, create a new project. You can name it whatever. Go into settings, enable web access. Then, enable Realtime Database, test mode. Go to settings, scroll to config, and copy the config and go to the code and replace the firebaseConfig const. Download, and share with friends.
